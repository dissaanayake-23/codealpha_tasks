Task 2: Automatic Light Detection System

Description

A sensor-based IoT simulation was developed using Arduino Uno, an LDR (Light Dependent Resistor), and an LED in Tinkercad. The system automatically controls the LED based on the surrounding light intensity.

Working Principle

* The LDR continuously measures ambient light levels.
* Arduino reads the sensor value through Analog Pin A0.
* When the environment becomes dark, the LED turns ON automatically.
* When sufficient light is detected, the LED turns OFF.

Components Used

* Arduino Uno
* Light Dependent Resistor (LDR)
* LED
* Resistors
* Breadboard
* Jumper Wires
* Tinkercad Simulator

Technologies Used

* Arduino IDE (C/C++)
* Tinkercad Circuit Simulator
